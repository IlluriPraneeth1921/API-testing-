2. Run Commands

🔥 Quick Smoke Test (2 scenarios per sheet)

npm run test:excel:smoke
cross-env MAX_SCENARIOS=2 SKIP_SQL=true npx playwright test excel-driven-e2e

📦 Full Regression (All 172 Excel files)

npm run test:excel
cross-env ENV_NAME=DevF1 npx playwright test excel-driven-e2e

🎯 Single Entity / File

cross-env EXCEL=609768_Appointment.xlsx npx playwright test excel-driven-e2e
cross-env EXCEL=Appointment npx playwright test excel-driven-e2e
cross-env TFS=609768 npx playwright test excel-driven-e2e

5. Debugging & Troubleshooting

IssueFix
No Excel files matchedCheck API-TestData/ folder has .xlsx files. Filename format: <TFS_ID>_<Entity>.xlsx
Cognito auth failedVerify AUTH_USERNAME/PASSWORD in env_config.json. Check network access to Cognito.
401 UnauthorizedToken auto-retries once. If persists, check credentials or user pool config in lib/auth.ts
Route not found (404)Add entity to config/route-registry.json or check Swagger auto-resolve
SQL connection failedUse SKIP_SQL=true or verify SQL_SERVER/SQL_DATABASE in env_config.json
Timeout errorsAuto-retries 3x with 3s delay. Increase timeout in playwright.config.ts
Variable not resolvedCheck SQL lookup keys. Use SKIP_SQL=true to bypass. Check lib/core/variable-resolver.ts
Files starting with xFiles prefixed with x are excluded (e.g. x647985_Guardianship.xlsx)
