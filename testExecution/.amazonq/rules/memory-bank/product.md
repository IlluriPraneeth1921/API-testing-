# Product Overview

## Project Purpose
pwd-bdd-ts is a Behavior-Driven Development (BDD) test automation framework built with TypeScript, Cucumber, and Playwright. It provides end-to-end testing capabilities for web applications with a focus on maintainability, readability, and multi-environment support.

## Key Features
- **BDD Testing with Cucumber**: Write tests in natural language using Gherkin syntax for better collaboration between technical and non-technical stakeholders
- **Playwright Integration**: Robust browser automation with cross-browser support (Chrome, Firefox, Safari)
- **Multi-Environment Support**: Configurable test execution across different environments (QA, STD-F1) using environment-specific configurations
- **Comprehensive Reporting**: Automatic generation of HTML reports with screenshots, test duration, and detailed execution results
- **Page Object Model**: Structured page objects and keyword-driven architecture for maintainable test code
- **TypeScript**: Type-safe test development with modern JavaScript features
- **Tag-Based Execution**: Run specific test suites using Cucumber tags (@smoke, @organization)

## Target Users
- QA Engineers and Test Automation Engineers building end-to-end test suites
- Development teams practicing BDD and requiring automated regression testing
- Organizations needing multi-environment test execution with detailed reporting

## Use Cases
- Automated regression testing for web applications
- Smoke testing for critical user journeys
- Organization and administration workflow validation
- Login and authentication testing
- Multi-environment deployment validation
